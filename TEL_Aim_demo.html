Autonomous SLAM Mobile Robot

Competition & Research Project Demo

Project Overview

This project demonstrates the development and implementation of an autonomous mobile robot using SLAM (Simultaneous Localization and Mapping) techniques. For more details, see the full report below.

Key Features

  • Autonomous navigation using SLAM
  • Real-time mapping and localization
  • Obstacle avoidance and path planning

Gallery

Technical Details

The robot utilizes a LIDAR sensor and ROS2 for real-time SLAM processing. More technical details and results are available in the downloadable report.

Results & Achievements

The robot successfully completed the competition tasks and received recognition for its robust SLAM implementation.
